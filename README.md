# **Проект "Блог"**

Стек: React, TypeScript, SCSS Modules, Vite

## Установка и запуск

1. Создайте в корне проекта файл `.env` с содержимым файла `.env.example`
2. Установите зависимости и запустите проект:

```
npm install
npm run dev
```
